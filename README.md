This Repository Contain the Implementation The EDBT 2026 Paper: **"Automating Efficient Data Collection through the Synergy of Agentic AI and Active Learning"**.


## 📁 Repository Structure

* `src/`: Core implementation of the **ExtendTab** algorithm and baseline models.
* `The evaluation scripts/`: Evaluation scripts used to reproduce the experimental results presented in the our technical report.
* `The full system demonstrations/`: Contains end-to-end demonstrations of our Agentic-System in action.
